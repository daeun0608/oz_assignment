CSS

웹 페이지의 스타일을 정의하는 언어 CSS(Cascading Stylesheet)
Cascading : "폭포같은", 계속되는, 연속적인
-> 여러 스타일이 한 요소에 적용될 수 있을 때, 우선순위 규칙과 적용 흐름이 "Cascading"


HTML은 다수의 태그로 CSS는 다수의 스타일 선언문으로 구성된다.


스타일 선언문
  선택자{
      속성명: 속성값;
   }




CSS적용(우선순위)

HTML에 CSS 적용
1. 인라인 스타일 (태그에 직접 기술) : 웹콘텐츠와 스타일시트를 분리하기 위해 사용 자제
2. 스타일 태그 추가 : 내부 스타일시트, 가능한 head에 기술
3. 문서 간의 연결 (HTML과 CSS연결) : head에서 link태그로 연결


1. 인라인 스타일 (태그에 직접 기술)

인라인 스타일은 곂쳐도 우선 적용

2. 스타일 태그 추가

곂칠경우 head에서 나중에 쓰여진 선언문으로 적용

3. 문서 간의 연결 (HTML과 CSS연결)
