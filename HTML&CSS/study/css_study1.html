웹 페이지의 스타일을 정의하는 언어 CSS(Cascading Stylesheet)
Cascading : "폭포같은", 계속되는, 연속적인
-> 여러 스타일이 한 요소에 적용될 수 있을 때, 우선순위 규칙과 적용 흐름이 "Cascading"


HTML은 다수의 태그로 CSS는 다수의 스타일 선언문으로 구성된다.


스타일 선언문
  선택자{
      속성명: 속성값;
   }

선택자

선택자의 우선순위
id > class > tag


전체선택자 ( * )
태그선택자 ( 태그명 )
클래스선택자 ( .클래스 )
아이디선택자 ( #아이디 )
그룹선택자 ( 선택자, 선택자, 선택자, ... )


- 속성 선택자 ( [속성명] )
    - *(포함 여부), ^(시작 여부), $(끝 여부)
    - ex) img[alt] / [alt] / [width $= "0"]
- 하위 요소 선택자 ( 공백 ) : 상위 요소에 포함된 하위 요소
- 자식 요소 선택자 ( > ) : 상위 요소의 바로 한 단계 하위 요소
- 인접 형제 선택자 ( + ) : 형제 요소 중 첫 번째 형제 요소
- 형제 요소 선택자 ( ~ ) : 앞은 포함하지 않고 뒤에 나오는 형제들을 대상

의사클래스

의사클래스(가상클래스)는 선택자에 추가하는 키워드 요소가 특정한 상태가 되었을 때 선택
ex ) h1:hover{ color:red; }

* 자주 쓰이는 의사 클래스
hover : 마우스 포인터가 올라감
active : 요소를 활성화/누르기
focus : 포커스상태 (ex. 입력창에 커서가 깜박거리는 바로 사용할 수 있는 상태)
disabled : 비활성화
nth-child() : 형제 사이에서의 순서에 따라 요소 선택자
    li:nth-child(2){ ... }
    li:nth-child(n){ ... } -> n은 자연수를 의미
    li:nth-child(2n){ ... }




의사요소

의사 요소는 요소의 특정 부분에 대한 스타일을 정의할 수 있다.
ex ) l1::first-letter{ font-size:20px; }

* 자주 쓰이는 의사 요소
after : 요소의 앞에 생성 및 추가
before
first-line : 블록 레벨 요소의 첫번째 선에 스타일 적용
marker : 목록 기호의 스타일 적용
placeholder : 입력요소에 자리표시자placeholder 스타일 적용
