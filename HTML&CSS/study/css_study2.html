박스모델

너비와 높이를 가지는 개별 요소를 '박스'로 표현하고, 박스를 4가지 세부 영역으로 나누어 구분한 일종의 디자인 모델
CSS로 정의할 수 있는 모든 요소는 박스 모델 관련 속성 중 일부 또는 전부가 적용




콘텐츠 영역

whdth, height > 주로 픽셀(px)나 % 단위를 사용
%는 자신이 속한 영역에 관련 값이 있어야 해서 기본 웹페이지 height는 정해져 있지 않아서 height를 %로 하면 적용X




테두리

border : 한번에 지정, 빈칸을 둘 수는 있음, 위치 상관X ( border-width, border-style, border-color )
border-radius : 좌측 상단부터 시계방향으로 설정할 수 있음






여백

테두리 안은 padding, 밖은 margin
- 여백은 속성값 개수에 따라 면들의 여백를 정할 수 있다.
(1) 모든면
(2) 위아래 / 왼오른
(3) 위 / 왼오른 / 아래
(4) 위 / 오른쪽 / 아래 / 왼쪽

- 접미사를 통해 여백면을 선택할 수 있다 ( top, bottom, left, right )
padding-top:20px;


margin: auto;너비에서 남는 여백을 동일 분배 box-sizing (기본 content-box)
내가 설정한 width/height가 padding과 border를 포함한 값인 'border-box'로 설정 > 콘텐츠가 작아짐

배경

background-color : 색코드
background-clip: border-box(기본) / paddig-box / content-box
background-image:url(이미지경로)
background-size:400px 400px
background-repeat: repeat(요소보다 작으면 기본 바둑판) / no-repeat / repeat-x / repeat-y
background-origin : border-box / padding-box(기본 이미지 시작) / content-box
background-position : 시작위치
     top / right / bottom / left / center
     top right(조합가능) / top left(기본값)
     100px 50px(x,y축 이동 가능)
