HTML

페이지의 뼈대를 만드는 언어 HTML ( 표준 HTML5 )
웹 페이지의 스타일을 정의하는 언어 CSS ( 표준 CSS3 )

HTML(HyperText Markup Language) : 웹페이지의 콘텐츠를 정의하기 위해 사용하는 언어
HyperText : 하이퍼링크를 통해 어떤 문서에서 다른 문서로 접근할 수 있는 텍스트
Markup : (콘텐츠를) 표시하다

html에는 표시될 화면 외에 '문서정보'가 포함
head : 문서의 정보를 기입하는 부분
body : 문서의 내용을 기입하는 부분

html 주의사항
1. 태그 이름은 소문자
2. 엔터를 이용한 줄바꿈 무시
3. 스페이스를 이용한 공백을 한번만 허용
4. 들여쓰기를 적절히 사용하는 것을 권장



TEXT
