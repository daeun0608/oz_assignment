컨테이너

콘텐츠나 레이아웃에 영향을 주지 않고, 단지 요소 여럿을 묶어 관리를 편하게 하는 역할을 하는 요소
콘텐츠 내용을 구분하거나 공통적인 스타일을 적용할 때 사용
div(블록), span(인라인)등의 태그를 사용




전역 속성

전역 속성 : 모든 html 태그에서 공통으로 사용하는 속성
id : 요소에 고유한 이름을 부여하는 식별자
class : 요소를 그룹 별로 묶는 식별자
style : 요소에 적용할 css 선언
title : 요소의 추가 정보를 툴팁으로 제공

컨테이너는 아무런 영향도 주지않고 코드만 구분한다. 전역속성는 모든 요소가 가진다.
