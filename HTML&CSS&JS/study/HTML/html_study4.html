메타데이터

헤드에 저장되는 html문서의 메타데이터(정보)
구성에 관여하지 않지만 정보를 제공하여 검색엔진이 검색할 때 참고하고 결과에 반영한다.

charset : 인코딩 정보 (설정하지 않으면 웹브라우저 설정 상황에 따라 자동인코딩하여 문자가 깨질 가능성이 생긴다.)
http-equiv : http(인터넷에서 데이터를 통신하는 프로토콜) 관련 정보를 지정하고, content속성에서 값 지정
name : 문서정보를 이름+값으로 제공해서 검색을 최적화함 (서비스에 중요)

IE 브라우저의 최신 버전 엔진을 사용

<meta http-equiv="x-ua-compatible" content="IE=edge">

페이지를 10초마다 새로고침

<meta http-equiv="refresh" content="10">

content 키워드 쉽표 구분함

<meta name="이름" content="값1, 값2">




시맨틱태그

시맨틱 : 의미론적인
시맨틱 태그 : 개발자와 브라우저에게 의미를 제공
- 목적/역할 표시 > 검색 엔진이 중요한 단서로 인식
- 음성 인식에 중요한 역할 > 웹 접근성 향상
- 코드의 가독성 향상
- 컨테이너의 남용을 방지

잘 쓰이는 header, main, footer + nav
