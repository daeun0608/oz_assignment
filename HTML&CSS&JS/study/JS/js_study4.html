이벤트

웹 프로그래밍에서 DOM에서 발생하는 다양한 액션 또는 상호작용 동작을 나타내는 프로그래밍 인터페이스
-> 버튼 클릭, 키보드 키다운, 입력폼 제출

이벤트 핸들러

이벤트가 발생되면 실행될 코드 블록 ( 주로 함수가 담당 )
const handleClick = function(){ // 이벤트 핸들러
    window.alert('환영합니다!')
}

const button = document.querySelector('button')
button.onclick = handleClick // 버튼의 이벤트 속성에 함수(핸들러) 대입
-> 함수를 대입하는 것과 호출하는것은 엄연히 다름, 소괄호 주의

button.onclick=function(){ // 이벤트 핸들러룰 악명함수로
    window.alert('환영합니다!')
}

addEventListener()

onclick, onkeydown과 같은 이벤트 속성 -> addEventListener 메소드 활용
- 추가한 이벤트 핸들러를 제거하는 메소드 존재
- 같은 타겟에 다수의 핸들러 등록 가능

addEventListener(이벤트명, 이벤트핸들러)
이벤트명 : click, keyup(키보드 누를 때마다), submit

이벤트 객채

추가적인 기능과 정보를 제공하기위해 이벤트 핸들러에 자동으로 전달되는 데이터
-> 핸들러에 매개변수를 추가해서 사용가능

- 이벤트 객체의 정보 확인
const handleClick = function(event){
    console.log(event)
}

- target 속성 이용
const handleClick = function(event){
    if(event.target.id=='span'){
        console.log('span!')
    }else if(event.target.id=='strong'){
        console.log('strong!')
    }
}

spanBtn.addEventListener('click', handleClick) ->인수는 사용 X
