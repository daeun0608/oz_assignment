window객체

객체

JS는 대부분 지시를 하고자하는 대상(객체)에게 입력 > 처리 > 출력 하는 절차를 따라 수행

객체.데이터 : 객체가 가지는 다양한 데이터 사용(속성)
객체.기능( ) : 객체가 가지는 다양한 기능을 수행(메소드)

window 객체

브라우저가 제공하는 최상위 객체
let, const가 아닌 모든 변수/함수는 자동으로 window 객체의 속성으로 등록된다 그래서 편의상 생략된다.

- alter
window.alert('경고창')
salter('명령문은 한줄 당 하나씩, 세미콜론을 사용하면 한줄에 여러개 가능')

- console
콘솔console은 브라우저의 디버깅 콘솔을 의미, 브라우저에 내장된 하위 객체
console.log('console 창에 log찍기')




식별자

let ( 변수를 만들겠다, 초기화하지 않고 선언만 가능 )
let 변수이름 = 데이터

const 상수이름 = 데이터 ( 선언과 초기화를 동시에 해야함, 초기화 하지않으면 값이 없는 것으로 고정 )




자료형

자료형 ( 숫자number, 문자열string, 불리언boolean, 다양한 목적과 형태의 객체 )
undefined : 데이터가 정의되지 않았음 ( 선언만 된 변수일 경우 )
null : 데이터가 없음

- 자료형 확인하기
console.log(typeof 3)

number

산술 연산 ( +, -, *, /, % ) + ( += 같은 산술대입 가능 )
문자 연산 : + 해서 새로운 문자열 생성

string

문자열 : 기호의 순차 수열로 문자, 숫자, 특수문자 등 다양한 기호를 조합해 만들 수 있는 '기호의 집합'
('', "", ``) -> 이후의 ``(템플릿 리터럴)는 포맷을 위해서 추가
