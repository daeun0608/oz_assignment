float

웹 요소들이 화면에 표시되는 순서는 기본적으로 html 문서 내에서 태그가 쓰여진 순서로 왼쪽 위에서 오른쪽 아래로 흐른다.
float속성은 요소를 문서의 흐름에서 제외하여 부모 또는 루트 요소의 기준으로 위치 변경
css3에서는 요소의 위치를 변경을 대체하는게 많아져서 사용률이 내려감

float : left, right, none(기본)

float를 적용한 요소인 '부동 요소'는 다 른 요소에 영향을 주는 역효과를 줄 수 있음 -> clear사용 clear : right, left, both

p태그인데도 마치 span처럼 작동 -> float의 역효과


clear로 정상작동




position

position으로 요소의 배치 방법을 결정 > tom / bottom / right / left가 최종 위치를 결정

position : static(기본) / relative / absolute / fixed
static : 문서의 흐름을 따름
relative : 문서의 흐름을 따른 자리를 기준으로 요소 이동
absolute : 상위 position속성을 가진 요소를 기준으로 이동, 없으면 화면 기준
fixed : 화면을 기준으로 고정
